Psytrance Kick and Bass Masterclass Point System.

How good are you at producing kick and bass? You will get 1 point for each competed task.

THE BABY STEPS:

  • ✅ Power on Apple, PC or Linux computer
  • ✅ Install a typical DAW like Logic, Ableton Live, Studio One, Reason, FL Studio, ProTools, Reaper etc.
  • ✅ Open a DAW
  • ✅ Install free vst synth
  • ✅ Open free vst in DAW
  • ✅ Set DAW and PC to 441000 Hertz 24 bit
  • ✅ Use an Audio interface for better quality and short latency.
  • ✅ Make sure the computer is using the Audio Interface, check with headphone or speakers.
  • ✅ Install the latest driver for your Audio Interface, typically from the manufacturer website.

THE BASICS

  • ✅ Use studio headphones in your Audio Interface
  • ✅ Get some studio monitors (Speakers made for music production).
  • ✅ Use acoustic foam treatment in your room (Cheap foam from amazon is better than nothing).
  • ✅ If you can afford Studio monitors and acoustic foam then invest in quality headphones instead.
  • ✅ Search the internet for free psytrance kicks and download them.
  • ✅ Search the internet for free psytrance bass and download them.
  • ✅ Set your DAW to a typical psytrance speed 138-150 BPM.
  • ✅ Make a simple half Bass.
  • ✅ Make a simple double bass.
  • ✅ Make a simple triple bass.
  • ✅ make a simple triplet bass.
  • ✅ Make a long half bass.
  • ✅ Crop bass audio samples and set sample volume to rolling triple Bass (kick, bass, bass, bass).
  • ✅ Make simple bass using any VST synth.
  • ✅ Experiment with different buffer samples setting for your Audio interface 8, 16, 32, 64, 128, 256, 512, etc.

THE ADVANCED

  • ✅ Use sidechain compression for rolling bass fx.
  • ✅ Crop and duplicate bass audio sample so they are the same length and not overlapping each other.
  • ✅ Set fade-out on bass audio sample to avoid clicks and pops.
  • ✅ Make rolling Bass using Velocity for volume change.
  • ✅ Make a Snappy Bass in any VST Synth by using the Low Pass Filter and Filter Envelope.

  • ✅ Make a Snappy double Bass.
